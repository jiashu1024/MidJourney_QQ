# Midjourney-QQ（停止维护）

基于QQ机器人框架[mirai](https://github.com/mamoe/mirai)搭建的Midjourney机器人

**由于没时间维护，该项目被我放弃，不再更新该项目，部署应该也无法使用了**

## 功能支持

- [x] 支持群聊和私聊

- [x] 支持Imagine,U，V指令,Imagine垫图
- [x] 支持配置阿里云翻译
- [x] 任务提交智能限流，尽量避免弹人机验证
- [x] 违规被删私发的作图信息也能获取
- [x] 用户鉴权，可配置qq角色，每种角色支持设置fast作图和relax作图模式的次数
